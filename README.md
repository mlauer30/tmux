# tmux
tmux config
## instructions
include tpm with this project 
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
create symlink for .tmux.conf
```
ln -s ~/<project-directory>/tmux.conf ~/.tmux/.conf
```
install after attaching to new session

```prefix``` ```ctrl+I```

save

```prefix``` ```ctrl+s```
